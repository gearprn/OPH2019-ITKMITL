กำหนดการกิจกรรมเปิดบ้านไอที 2019

  อาคารเรียนรวมและปฏิบัติการคณะเทคโนโลยีสารสนเทศ สถาบันเทคโนโลยีพระจอมเกล้าเจ้าคุณทหารลาดกระบัง

  • 23 สิงหาคม
  • 24 สิงหาคม
 กำหนดการวันศุกร์ที่ 23 สิงหาคม 2562
08.00 - 09.00 น.

ลงทะเบียน

เริ่มลงทะเบียนเข้าร่วมงานและการแข่งขันวิชาการ ของโรงเรียน นักเรียน และประชาชน ผู้สนใจทั่วไป

สถานที่ : บริเวณจุดลงทะเบียน ณ อาคารคณะเทคโนโลยีสารสนเทศ

09.00 - 16.00 น.

นิทรรศการทางวิชาการ

นิทรรศการแสดงผลงานนักศึกษา และให้ความรู้ทางด้านเทคโนโลยีสารสนเทศ

กลุ่มเป้าหมาย : นักเรียน นักศึกษา และบุคคลทั่วไปที่สนใจ
สถานที่ : บริเวณชั้น 1 ภายในอาคารคณะเทคโนโลยีสารสนเทศ

11.00 - 12.00 น.

เสวนาหลักสูตร

  • แนะนำคณะฯ และหลักสูตร
  • แนะนำเรื่องการสมัคร TCAS เข้าคณะไอที
  • ทุนการศึกษา หอพัก ชุมนุม และสภาพแวดล้อม
  • แนะนำวิชาชีพและงานด้านไอทีโดยคณาจารย์และนักศึกษาของคณะ

กลุ่มเป้าหมาย : นักเรียนระดับมัธยมศึกษาตอนปลาย, และบุคคลทั่วไปที่สนใจ, ผู้ปกครองสมัครผ่านเว็บไซต์
สถานที่ : ห้องประชุมใหญ่ (Auditorium) อาคารคณะเทคโนโลยีสารสนเทศ

10.00 - 15.00 น.

กิจกรรม IT Career Workshop

สัมผัสประสบการจริงในการเป็นนักไอที แนะนำโดยอาจารย์และนักศึกษารุ่นพี่ เช่น การทำ workshop สาขาวิชาเทคโนโลยีสารสนเทศ แบ่งเป็น 3 แขนงวิชา คือ

  • แขนงวิชาการพัฒนาสื่อประสมและเกม กิจกรรม สายลับจับผิดภาพ
  • แขนงวิชาวิศวกรรมซอฟต์แวร์ กิจกรรม Software Defined Robots
  • แขนงวิชาเทคโนโลยีเครือข่ายและระบบ กิจกรรม IoT Connecting Things : เชื่อมต่อทุกสิ่งด้วย IoT
  • สาขาวิชาวิทยาการข้อมูลและการวิเคราะห์เชิงธุรกิจ กิจกรรม ตามหาโปเกม่อน ด้วยวิทยาการข้อมูล (จัดเป็นรอบๆ รอบละ 30 - 40นาที)

กลุ่มเป้าหมาย : นักเรียนระดับมัธยมศึกษาตอนปลาย
สถานที่ : ห้องบรรยาย M03, M04, M21 และ M22 อาคารคณะเทคโนโลยีสารสนเทศ

09.00 - 15.00 น.

การแข่งขันแก้ปัญหาเชิงวิเคราะห์ (Data Analysis)

การแข่งขันเพื่อส่งเสริมและสนับสนุนให้นักเรียนให้ความสนใจในการคิดเชิงตรรกะ การ คิดเชิงตัวเลข และการแก้ปัญหาเชิงวิทยาการข้อมูลด้วยการวิเคราะห์ได้อย่างเหมาะสม

กลุ่มเป้าหมาย : นักเรียนระดับมัธยมศึกษาตอนปลายโดยสมัครเข้าร่วมในนามโรงเรียน
สถานที่ : ห้องปฏิบัติการคอมพิวเตอร์ 205 ชั้น 2 อาคารคณะเทคโนโลยีสารสนเทศ

09.00 - 15.00 น.

การแข่งขันกีฬาอิเล็คทรอนิกส์ E–sports

การแข่งขันเพื่อให้ผู้เข้าแข่งขันได้มีโอกาสแสดงความสามารถในกีฬาอิเล็คทรอนิกส์ (Esports) เป็นการฝีกการทำงานร่วมกันเป็นทีม และเพื่อสร้างแรงบันดาลใจให้ผู้เข้าชม อยากทำการพัฒนาและมีส่วนร่วมในการเล่นเกมให้เกิดประโยชน์

กลุ่มเป้าหมาย : นักเรียนระดับมัธยมศึกษาตอนปลายโดยสมัครเข้าร่วมในนามโรงเรียน
สถานที่ : ห้องบรรยาย 226 ชั้น 2 อาคารคณะเทคโนโลยีสารสนเทศ

09.00 - 15.00 น.

การแข่งขันด้านความปลอดภัย ของระบบคอมพิวเตอร์ (Network security)

การแข่งขันเพื่อนักเรียนได้มีความรู้ ความสามารถ และรู้ทันภัยคุกคามในอินเทอร์เน็ต และในระบบคอมพิวเตอร์ อีกทั้งยังนำความรู้ไปใช้ให้เกิดประโยชน์ในด้านการป้องกัน ตนเองและผู้อื่นจากภัยคุกคามในอินเทอร์เน็ตและในระบบคอมพิวเตอร์ได้

กลุ่มเป้าหมาย : นักเรียนระดับมัธยมศึกษาโดยสมัครเข้าร่วมในนามโรงเรียน
สถานที่ : ห้องปฏิบัติการคอมพิวเตอร์ 203 ชั้น 2 อาคารคณะเทคโนโลยีสารสนเทศ

09.00 - 15.00 น.

การแข่งขันพัฒนาเว็บไซต์ด้วย PHP และ JavaScript

การแข่งขันเพื่อสนับสนุนส่งเสริมเยาวชนให้มีความรู้ความสามารถในการพัฒนาเว็บไซต์ และทักษะการเขียนโปรแกรมด้วยภาษา PHP และ JavaScript

กลุ่มเป้าหมาย : นักเรียนระดับมัธยมศึกษาโดยสมัครเข้าร่วมในนามโรงเรียน
สถานที่ : ห้องปฏิบัติการคอมพิวเตอร์ 207 ชั้น 2 อาคารคณะเทคโนโลยีสารสนเทศ

13.30 - 14.30 น.

ทักษะการคิดเชิงคำนวณ (computational thinking skill)

  • การทำแบบทดสอบทักษะการคิดเชิงคำนวณ (สาหรับนักศึกษา)
  • เชิญชวนครูแต่ละโรงเรียนร่วมจัดการทดสอบทักษะการคิดเชิงคำนวณนานาชาติ (Bebras Challenge)

กลุ่มเป้าหมาย : นักเรียนระดับมัธยมศึกษาตอนปลาย, ครูฝ่ายแนะแนวและครูสอนคอมพิวเตอร์
สถานที่ : ห้องปฏิบัติการคอมพิวเตอร์ 308 ชั้น 3 และห้อง 231 ชั้น 2 อาคารคณะเทคโนโลยีสารสนเทศ

09.00 - 16.00 น.

พบปะรุ่นพี่เตรียมความพร้อมสู่ไอทีลาดกระบัง “ToBeIT@KMITL”

แนะนำเกี่ยวกับการเตรียมตัวสู่รั้วมหาวิทยาลัย และการเตรียมสอบเข้า คณะเทคโนโลยีสารสนเทศ ลาดกระบัง โดยรุ่นพี่นักศึกษาที่กำลังศึกษาอยู่

กลุ่มเป้าหมาย : นักเรียนระดับมัธยมศึกษาตอนปลาย
สถานที่ : ห้อง M23 ชั้นล่าง

10.00 - 16.00 น.

เยี่ยมชมคณะฯ

เยี่ยมชม ห้องปฏิบัติการคอมพิวเตอร์ ห้องบรรยาย ห้อง Server, common room (ห้องสันทนาการ) ชุมนุมต่างๆ ของนักศึกษา ฯลฯ

สถานที่ : บริเวณหน้าห้องประชุมใหญ่ (Auditorium)

10.00 - 15.00 น.

กิจกรรมจากชุมนุมต่างๆ ของนักศึกษาในคณะ

  • ชุมนุมสันทนาการ (AGAPE)
  • ชุมนุม PHOTO (ถ่ายรูป)
  • ชุมนุม INFONIC (ดนตรี)
  • ชุมนุม อนิเมะและเกมกระดาน (เกม)

สถานที่ : ห้องบรรยาย M 14, M 15, M 17 และห้อง Common Room ชั้นล่าง
